Simple sharing plugin to display social shared count from facebook and twitter

== Description ==

Wp Simple Share will display facebook and twitter buttons along with social shared count on singular posts:

* Posts.

<p>After you install, goto wordpress admin panel -> Settings -> Wp Share Simple </p>
<p>-> Choose the display option</p>
<p>-> Enter Hex color code for share count </p>
<p>-> Save Settings </p>

== Installation ==

The easiest way to install wp share simple is :
<p>Go to your website admin screen plugins page </p>
<p>Click on Add New </p>
<p>Search for  'Wp Share Simple'  and install</p>


Manual Installation
<p>1. Upload the `wp-share-simple` folder to the `/wp-content/plugins/` directory</p>
<p>2. Activate the plugin through the 'Plugins' menu in WordPress</p>


== Frequently Asked Questions ==

= Where is the options page ? =

  Go to Settings >> Wp Share Simple

= How to Activate the plugin ? = 
  
  Go to Settings >> Wp Share Simple >> Check the option Active Simple Share Buttons and Save
