Simple sharing plugin to display social shared count from facebook and twitter

== Description ==

Wp Simple Share will display facebook and twitter buttons along with social shared count on singular posts:

* Posts.

== Installation ==

The easiest way to install wp share simple is :
Go to plugins page 
1. Add New 
2. Search for  'Wp Share Simple' 
3. Find it by username 'mannemvamsi' and install.

Manual Installation
1. Upload the `wp-share-simple` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
